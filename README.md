## 🔥 Rules 🔥
    * 주 5일 이상 기록한다.
    * 의미 없는 기록과 커밋은 금지한다.
    * 프로젝트를 진행하지 않은 날에는 필수로 기록한다.
    * 하루의 기준은 새벽을 포함한 '자기 전'까지이다.
    * 월요일엔 일주일간의 목표를 세우고, 일요일엔 되돌아보는 시간을 가진다.
    * 모든 목표엔 우선순위를 매긴다.
    * 부담가지지 않는다. 공부한 만큼만 기록한다.
